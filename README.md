 Issue Tracker

A responsive and dynamic web application to track, filter, and manage issues efficiently. This project features a secure login system and an intuitive dashboard to view issue details based on their current status.

**🔗 https://mrnafi1.github.io/assignment-5/

---





Key Features

- **Secure Login:** Access the dashboard using demo credentials.
- **Dynamic Dashboard:** A fully responsive 4-column card layout for displaying issues.
- **Advanced Filtering:** Easily filter issues by **All**, **Open**, and **Closed** status using tab navigation.
- **Search Functionality:** Quickly find specific issues using the integrated search bar.
- **Detailed Modal View:** Click on any issue to view in-depth details in a dynamic modal without leaving the page.
- **Status-based Styling:** Visual cues for issue status (e.g., Green borders for 'Open' and Purple for 'Closed' issues).
